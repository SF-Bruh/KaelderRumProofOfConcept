# KælderRum — Basement Storage Marketplace

A proof-of-concept website for **KælderRum**, an Airbnb-like platform connecting homeowners with unused basement space to people who need affordable, secure storage in Denmark.

## Getting Started

Open `index.html` in any modern browser — no build tools or dependencies required.

## Structure

- `index.html` — Single-page website with all sections
- `styles.css` — Premium styling with glass-morphism, animations, and responsive layout
- `script.js` — Scroll reveals, typing effect, counter animations, form validation

## Sections

- **Hero** — Bold headline with typing animation and live stats
- **About Us** — Service overview and founder information
- **The Product** — Feature showcase (security, insurance, booking, verification, flexibility)
- **How It Works** — Step-by-step flows for renters and hosts
- **Pricing** — Three tiers: Basic, Premium, Business (DKK)
- **Create Account** — Registration form with live validation and password strength indicator
